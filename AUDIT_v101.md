# AUDIT v101

Corecții:
1. Modalul a fost refăcut integral, nu doar corectat prin CSS peste varianta veche.
2. Clasele sunt noi și izolate: `svt-help101-*`.
3. Inputurile facturii nu depind de `:has()`.
4. Etichetele sunt sparte explicit cu `<br>`.
5. Taburile sunt operate cu un listener nou și robust.
